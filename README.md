## 2019年Esri开发大赛 《冬奥小助》
# 1.作品介绍：
这是一款为北京2022冬奥会志愿者，运动员以及观众提供帮助的App,包含赛区预览，赛区信息，场馆介绍，项目介绍，室外和室内导航，室内地图（由于大部分场馆仍在建设中，目前只做了水立方的室内地图），安全应急疏散，社区交流与分享，短视频资讯，赛事信息个性化推荐，云端管理，志愿者调配，人流统计与分析等功能。
